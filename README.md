## Game Charts
This is a simple game made by using JavaScript and HTML5 canvas. Scores are stored in a Firebase Realtime Database and are used to create canvas charts by using a Chart.js library. 
<br><br> The first chart shows the top 5 scores. The second chart shows averages and how many scores have the same value. You can toggle the chart by clicking its label.
            
### Gameplay   
Press SPACE to hover and try to avoid obstacles. You score when obstacles reach the other side. The game ends when you hit the obstacle and your score is then sent into the database.
